# Hokus — идея проекта

**Hokus** — это приложение для фокусировки и спокойной работы через персональные атмосферные звуки.

Главная идея: у каждого пользователя есть NFT-профиль, который определяет его уникальный звуковой ландшафт. Один NFT может звучать как мягкий дождь, другой — как дождь с редкими раскатами грома, третий — с дополнительными звуками природы. Таким образом, каждый профиль становится не просто цифровым объектом, а личной аудио-атмосферой.

Пользователь может создать NFT прямо в приложении и сразу использовать его как источник звука для концентрации, учебы, отдыха или медитации.

## Что делает Hokus особенным
- Фокус не на «музыке из плейлиста», а на **персональном звуке**, связанном с NFT.
- У каждого NFT свой характер и свой ритм.
- Это сочетание практической пользы (концентрация) и коллекционной ценности (уникальные профили).

## Пользовательский путь
1. Пользователь открывает приложение.
2. Подключает кошелек.
3. Создает NFT-профиль или выбирает уже существующий.
4. Запускает звук и использует его как фон для фокусировки.
5. Переключает NFT, если хочет другую атмосферу.

## Визия
Hokus — это «саундтрек твоего состояния», где каждый NFT отражает настроение и стиль фокуса пользователя.
